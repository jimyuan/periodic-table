# Periodic Table 元素周期表
为纪念元素周期表第七周期元素全部填满！

## 源码运行

``` bash
# install dependencies
# 安装依赖（仅首次运行需执行）
npm install

# serve with hot reload at http://0.0.0.0:8080
# 默认在 8080 端口开启热加载 web server
npm start

# build for production with minification
# 编译成生成环境所需代码
npm run build
```
